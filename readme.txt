== Description ==

Providing an easy way for a live chat widget integration for [LightChat](https://lightchat.org/en/) customers.
Using the plugin you can easy setup target pages which will display the widget.
The plugin is supporting multilingual pages.
For using the plugin you need have an account in [live chat](https://lightchat.org/en/) LightChat.org.
After sign up you will be able to setup a widget design and get it code, after you need set the code in a plugin settings.
After it the widget will be displaying on your website target pages and you will be able to have a chat with go to [special page](https://lightchat.org/en/chat/) in the service. This way doesn't require any additional actions and your website modification. The widget is adaptive for different devices (pc/tablet/mobile).
[Videoreview](https://www.youtube.com/watch?v=8jnmloEM6h4), [more about the service](https://lightchat.org/en/).

== Installation ==

1. Install and activate the plugin
2. Go to a `Settings - Light Chat widget` page
3. Modify the settings and save
4. Widget will appear on the target pages - that's all
